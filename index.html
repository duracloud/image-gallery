Edit Collections

How to Edit Collections

  • A Collection respresents a group/collection of images within a DuraCloud Space.
  • Images within a collection must be organized using the following structure:
    • thumbs (thumbnail images displayed in a grid, all viewable on the page)
    • previews (larger versions of the thumbnail images, displayed when a thumbnail is clicked)
    • highres (high resolution images that can be viewed by clicking on the link underneath the blownup preview images)
  • Example organization structure:
    • collection1/thumbs/img1.jpg
    • collection1/thumbs/img2.jpg
    • collection1/previews/img1.jpg
    • collection1/previews/img2.jpg
    • collection1/highres/img1.jpg
    • collection1/highres/img2.jpg
    • collection2/thumbs/img1.jpg
    • collection2/thumbs/img2.jpg
    • collection2/previews/img1.jpg
    • collection2/previews/img2.jpg
    • collection2/highres/img1.jpg
    • collection2/highres/img2.jpg
  • A collection can have any name. The above 'collection1', 'collection2' are arbitrary examples.
  • The images in a collection can have any name, but the corresponding images between thumbs, previews, and highres must have the same name. The above 'img1.jpg', 'img2.jpg' are arbitrary examples.
  • Spaces can be added by clicking the '+' button. The space must already exist within DuraCloud.
  • Spaces can be removed by selecting the space to remove from the dropdown menu and then clicking the '-' button.
  • Adding/Removing collections from a space:
    • Select the space to add/remove collections to/from.
    • The current collections will be displayed in the text area below the selected space.
    • To add a collection, type in the name of the collection on a new line.
    • To remove a collection, delete the line containing the name of the collection.
  • Edits are not saved until the 'Save' button is clicked.

DuraCloud Image Gallery Tips & Tricks

A few tips and tricks about the DuraCloud image gallery:

  • When the page is loaded the available spaces and image collections are retrieved and loaded into the dropdown menu. Selecting a space from the dropdown menu triggers the thumbnails in that space to be loaded.
  • If this is your first time using the image gallery you will most likely not have any spaces or collections availble to select from the dropdown menus. You can set up your available spaces and image collections by clicking the "Edit" button at the top of the screen. There is a "(?) Help" button on the Edit window explaining how to setup your available spaces and image collections.
  • The thumbnails are displayed in a grid with the name of each image underneath its thumbnail.
  • Clicking a thumbnail will load a larger version of the image in the foreground.
  • When viewing a blownup thumbnail:
    • The blownup image can be zoomed in and out by scrolling on it (mousewheel or trackpad).
    • A zoomed in image can be panned around by moving the mouse around.
    • Navigating to the next/previous image can be done by clicking the left or right arrows on either side of the screen or by pressing the left or right arrow keys on the keyboard.
    • You can exit from viewing the blownup image display by clicking the 'X' at the top-right corner of the screen or by pressing the "Esc" or "x" key on the keyboard.
    • Clicking the content ID link below the blown up image loads the high-res jpg in a new browser tab/window.
  • Most of the above functionality works the same on mobile devices.
    • Swipe left/right to move to previous/next image.
